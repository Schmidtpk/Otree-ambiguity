Wir betrachten ob ein roter oder blauer Ball gezogen wird. Sie können in dieser Runde 2€ gewinnen. Sie müssen eine von drei Möglichkeiten wählen:

  1. Sie erhalten 25 Punkte sicher, egal welche Farbe gezogen wird
  2. Sie erhalten 50 Punkte, nur falls ein roter Ball gezogen wird
  3. Sie erhalten 50 Punkte, nur falls ein blauer Ball gezogen wird

Nachdem Sie ihre Wahl getroffen haben, werden wir den Ball ziehen.

Wenn Sie Option 1 gewählt haben, erhalten Sie sicher 25 Punkte, egal was für ein Ball gezogen wird. Das heißt Sie haben eine 25% Chance auf die 2 €. Diese Option ist also unabhängig von der Unsicherheit der Ballfarbe.

Wenn Sie Option 2 gewählt haben, und ein roter Ball gezogen wird, erhalten Sie 50 Punkte. In diesem Fall haben Sie also eine 50% Chance auf die 2 €. Sollte ein blauer Ball gezogen werden, erhalten Sie keine Punkte. In diesem Fall haben Sie also keine Chance auf die 2 €.

Wenn sie Option 3 gewählt haben, und ein blauer Ball gezogen wird, erhalten Sie 50 Punkte. In diesem Fall haben Sie also eine 50% Chance auf die 2 €. Sollte aber ein roter Ball gezogen werden, erhalten Sie keine Punkte. In diesem Fall haben Sie also keine Chance auf die 2 €.

Es gibt keine richtigen oder falschen Antworten, wir möchten ihre persönliche Einschätzung der Unsicherheit erforschen.
